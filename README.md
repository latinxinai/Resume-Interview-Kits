## Quickstart

1) **Clone or download** this repo.  
2) **Pick your role track** (General / ML Eng / AI Research) in [`/prompt-bank/`](./prompt-bank/).  
3) **Create a role-specific resume** using the ATS template and the JD alignment prompts.  
4) **Spin up a project** (or package an existing one) using the Portfolio README + Case Study.  
5) **Practice interviews** with the STAR/interview panel prompts.  
6) **Track applications** in the Excel/CSV tracker; review your funnel weekly.  
7) **Negotiate cleanly** using Script **A** *or* **B** (don’t mix).
